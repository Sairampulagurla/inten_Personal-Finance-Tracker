# 💰 Personal Finance Tracker

## Project Overview

Personal Finance Tracker is a web-based application that helps users manage their personal finances by tracking income, expenses, savings, and financial reports. The application provides an interactive dashboard, transaction management system, and visual analytics through charts.

## ✨ Features

### Dashboard
- Total Income
- Total Expenses
- Current Balance
- Total Transactions

### Transaction Management
- Add Income Transactions
- Add Expense Transactions
- Delete Transactions
- Search Transactions
- Transaction Categories

### Reports
- Financial Summary
- Savings Calculation
- Interactive Pie Chart

### Additional Features
- Dark Mode
- Client-side Routing
- Local Storage Support
- Responsive Design
